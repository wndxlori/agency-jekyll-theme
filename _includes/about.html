About

The WNDX Group Inc. has been in operation for 20 years.

  • 1996

    Early Days

    The WNDX Group was originally formed to continue supporting and developing the WNDX GUI Toolkit, a set of cross-platform C libraries, used to create cross platform graphical windowing user interfaces for applications on DOS, Windows, X-Windows (Sun Solaris, HP-UX, SGI Irix), and Macintosh.

  • 1999

    Brewing some Java

    When Java came along, and kicked the crap out of the cross-platform business. If you can't beat'em, join'em. WNDX became a Java programming consultancy, specializing in early applets for the web, desktop applications, and finally enterprise web applications.

  • 2007

    Rocking the Rails

    Introduced to Ruby on Rails pre-1.0, we were so impressed with the boost in productivity it provided, that we immediately started our next project with Ruby on Rails, and continue to develop Rails application for appropriate business solutions today.

  • 2011

    Passion for Teaching

    After many years of informally mentoring and coaching the junior developers on the teams with which we have worked, it was time to formalize some of that knowledge, and we created the Ruby on Rails for Real Developers series of workshops.

  • 2014

    Author! Author!

    Ruby the language was developed to make programmers happy. It made us very happy when RubyMotion was released, allowing us to develop iOS applications using Ruby, instead of Objective C. The result... we started developing mobile applications, and our first book was written and self-published ‐ Core Data in Motion.
